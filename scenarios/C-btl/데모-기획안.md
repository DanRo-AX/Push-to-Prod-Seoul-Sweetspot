# 달무드 「문라이트」 팝업 기획안 (제안용)

> 프로젝트: 달무드 첫 단독 팝업 「문라이트」 시향 전시
> 기간: 2026.06.24(수)~06.28(일), 12:00–20:00 / 장소: 성수동 연무장길 단독 공간(약 40평)
> 작성: 팝가 BTL 기획팀 · 제안 단계 초안 v1

## 1. 핵심 메시지

**"하루의 끝, 달이 뜨는 시간을 당신의 방으로."**

핸드포어드 캔들 달무드의 첫 오프라인 거점. '향'을 눈으로 보고 손으로 붓는 경험으로 번역해,
무명 브랜드의 첫인상을 '차분하고 깊은 브랜드'로 각인한다.

## 2. 컨셉 — 하룻밤의 동선

3개 라인을 하룻밤의 시간대로 공간화한다. 관람객이 초승→보름→그믐 순으로 통과한다.

| 구간 | 라인 | 공간 무드 |
| --- | --- | --- |
| 입구 | 초승 「새벽 정원」 | 청귤·배꽃, 산뜻한 시작. 낮은 조도 |
| 중앙 | 보름 「월하향」 | 시그니처. 달빛 조명 + 향의 절정 |
| 안쪽 | 그믐 「깊은 밤의 숲」 | 시더우드·바닐라, 잠들기 전 어두운 톤 |
| 2층 | 포어링 시연존 | 캔들을 직접 붓는 과정 시연 |

전체 톤은 정숙·감상 위주의 갤러리. 각 구간의 '한 컷'을 어디서 어떻게 담아 공유하게 할지는
별도 설계 없이 관람객 자율에 맡긴다.

## 3. 문라이트 시향 세션 (핵심 프로그램)

- 회당 6명, 25분, 3라인 순서 시향 + 미니 캔들 포어링 체험.
- **100% 사전 예약제 — 홈페이지 회원가입 후 예약.** 워크인(현장 방문)은 받지 않는다.
- 예약자 전원에게 「새벽 정원」 티라이트 증정.

## 4. 온라인 확산

- 공간·제품 키비주얼 촬영, 60초 무드 영상 제작.
- 라이프스타일 인플루언서 초청, 후기 콘텐츠 위탁.
- 관람객이 직접 만드는 콘텐츠(인증·릴스)에 대한 장치나 유인은 아직 없다.

## 5. 현장 한정

- 팝업 에디션 라벨(배치 번호·날짜 수기 기입) 연출.
- 별도의 현장 한정 판매나 재방문 후크는 두지 않는다.

## 6. 일정·예산(요약)

- 착수 → 공간 확정 → 시공(6/22~23) → 운영(6/24~28) → 철거(6/29).
- 총액 VAT 포함(항목별 견적 별첨). 공간·시공이 비중의 절반 이상.

## 7. 성공 지표

- 시향 세션 예약 건수·참석률, 인플루언서 도달, 영상 조회수.
